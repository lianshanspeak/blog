# git知识扫盲

![image-20220909103140349](./image/image-20220909103140349.png)



> git status	 查看库当前状态有变更文件
>
> git diff	比较文件的不同，即暂存区和工作区的差异。
>
> git log 查看历史提交记录
>
> git add . - 添加文件到暂存区
>
> git commit - 将暂存区内容添加到仓库中
>
> git reset	回退版本
>
> git push 推送到远程

自己操作步骤

git status 看状态有没有改的

git add . 提交本地所有文件到缓冲区

git commit -m "做了什么操作"

git push 推送到远程
